# Jos Marketplace

A modern, professional marketplace website built with Next.js, React, Framer Motion, and Tailwind CSS.

## Features

- Animated hero section with search functionality
- Category browsing with smooth animations
- Product grid with hover effects
- Responsive design
- Professional UI/UX with Framer Motion animations
- Sticky navigation bar
- Footer with social links

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Framer Motion
- Tailwind CSS
- Lucide React Icons
